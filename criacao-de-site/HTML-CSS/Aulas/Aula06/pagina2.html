Interação com o usuário

Seletor de cor

O seletor de cor permite ao usuário escolher uma cor de forma interativa diretamente na página. Ele é criado utilizando a tag input com o tipo "color", que abre uma paleta de cores. Esse recurso é bastante utilizado em formulários e sistemas modernos. Para criar um seletor de cor no HTML, utilize a tag input type="color". Ela gera um seletor visual nativo do navegador, permitindo ao usuário escolher a cor e retornando o valor em formato hexadecimal

Links

Como criar links

Os links são fundamentais para a navegação na internet, permitindo acessar outras páginas. Eles são criados com a tag a, utilizando o atributo href para definir o destino. Também é possível abrir o link em nova aba com o atributo target="_blank".
