HTML e CSS

HTML e CSS são duas das principais tecnologias utilizadas no desenvolvimento web moderno. O HTML é responsável por estruturar o conteúdo da página, organizando textos, imagens e elementos. Já o CSS é utilizado para melhorar a aparência, definindo cores, fontes e espaçamentos.

Como usar títulos no HTML

Os títulos são criados com as tags h1 até h6, sendo o h1 o mais importante dentro da página. Eles ajudam a organizar o conteúdo em seções, facilitando a leitura e o entendimento. Além disso, são importantes para a estrutura do site e para mecanismos de busca.

Como criar listas

As listas são utilizadas para organizar informações de maneira clara dentro de uma página. Para criá-las usamos a tag ul para listas simples e li para cada item. Também é possível criar listas dentro de outras listas, chamadas listas aninhadas.

Como criar uma tabela

As tabelas são usadas para organizar dados em linhas e colunas de forma visual. Para criá-las utilizamos a tag table, com tr para linhas e th ou td para células. Também podemos usar colspan e rowspan para juntar células e melhorar a organização.

Conclusão

O HTML é uma linguagem essencial para a criação de páginas web, permitindo estruturar conteúdos de forma organizada. Ao longo deste trabalho, foi possível compreender como utilizar títulos, listas, tabelas e links na prática. Esses conhecimentos são fundamentais para quem deseja iniciar na área de desenvolvimento web.

Além disso, os exemplos e vídeos apresentados ajudam a reforçar o aprendizado. Com a prática, é possível evoluir e criar páginas cada vez mais completas e interativas. Dessa forma, o HTML se torna a base para qualquer desenvolvedor web.
